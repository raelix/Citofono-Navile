Citofono-Navile Beta 1.0
---------------

Servizio per l'accesso rapido all'applicazione SkyBell (per utenti inesperti). Pulsanti di stato per attivare/disattivare la videochiamata. Possibilità di accensione automatica tramite riconoscimento del suono del campanello (con la possibilità di scegliere una soglia) e di disattivazione automatica con riconoscimento del movimento tramite fotocamera anteriore. Le automazioni vengono svolte tramite API di console native quindi è necessario l'uso di uno Smartphone android rootato. Attualmente in uso come video citofono in una palazzina di 9 appartamenti, dotati di tablet Android.
